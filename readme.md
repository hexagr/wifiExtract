# readme

A small program written in C++ to enumerate and extract WLAN profiles on Windows.

```
>msbuild wifiExtract\wifiExtract\wifiExtract.vcxproj

...

>wifiExtract.exe
[+] Found Wifi Profile:
[+] SSID: openNet
[+] No key material found

[+] Found Wifi Profile:
[+] SSID: networked
[+] Key: password123
```
